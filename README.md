# Hope Theory HQ

> Edge. Resilience. Never GIVE UP.

A premium personal brand website built with React 19, Vite 7, TypeScript 5.9, and Tailwind CSS 3.4.

## Tech Stack

- **Frontend:** React 19.2 + TypeScript 5.9
- **Build:** Vite 7.2
- **Styling:** Tailwind CSS 3.4 + custom design tokens
- **Animation:** Framer Motion 12.34 + Lenis smooth scroll
- **UI Components:** 40+ shadcn/ui (Radix) primitives
- **Particles:** tsparticles

## Design Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `ht-navy` | `#0a0e17` | Background |
| `ht-green` | `#00ff9d` | Primary accent |
| `ht-amber` | `#fbbf24` | Secondary accent |
| `ht-gold` | `#d4af37` | Tertiary accent |

## Getting Started

```bash
npm install
npm run dev
```
